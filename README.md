## How to use 

1. Get your anilist api token: [Anilist Api](https://anilist.gitbook.io/anilist-apiv2-docs)

2. Add your anilist API token to variable: [ani_pie_char.py](ani_pie_chart.py)

3. Run python file: **ani_pie_chart.py**
